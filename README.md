### Installation

#### CentOS/RHEL 6.x
```
[sudo] yum install -y https://yum.kaos.io/6/release/i386/kaos-repo-7.0-0.el6.noarch.rpm
```

#### CentOS/RHEL 7.x
```
[sudo] yum install -y https://yum.kaos.io/7/release/x86_64/kaos-repo-7.0-0.el7.noarch.rpm
```
